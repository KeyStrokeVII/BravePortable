============================================
        BRAVE PORTABLE - Version 1.0
============================================

Welcome to Brave Portable! This is a fully portable version of Brave Browser
that requires no installation and leaves no traces on your system.

GETTING STARTED
===============
1. Double-click "BravePortable.exe" to launch Brave
2. Your profile data is stored in the "Data" folder
3. Pre-loaded extensions are in the "Extensions" folder

FOLDER STRUCTURE
================
BravePortable.exe  - Main launcher (run this)
config.json        - Configuration file
Browser/           - Brave browser application files
Data/              - Your profile, bookmarks, and settings
Extensions/        - Pre-loaded extensions (Bitwarden included)

FEATURES
========
✓ Fully portable - run from any drive or USB stick
✓ Automatic updates - checks for new Brave versions daily
✓ No registry entries - completely self-contained
✓ Pre-loaded Bitwarden password manager extension
✓ All your data stays in the Data/ folder

UPDATING
========
The launcher automatically checks for Brave updates daily.
When an update is available, it will download and install automatically.

ADDING MORE EXTENSIONS
======================
1. Download CRX files from Chrome Web Store using:
   https://clients2.google.com/service/update2/crx?response=redirect&prodversion=98.0&acceptformat=crx3&x=id%3D{EXTENSION_ID}%26uc
2. Place CRX files in the Extensions/ folder
3. Restart Brave - extensions will be extracted and loaded automatically
